> Cortex is a local-first AI memory layer designed for agents that need to remember.

It gives AI systems persistent memory across sessions, allowing them to store and recall decisions, mistakes, preferences, and long-term context instead of starting from scratch every time.

Cortex is built for agentic workflows where continuity matters — chatbots, coding agents, autonomous tools, and experimental AI systems.

---

## 🧠 What Cortex Does

- Stores long-term memory for AI agents  
- Tracks decisions, mistakes, and learned behavior  
- Restores context across sessions automatically  
- Helps agents improve over time instead of resetting  

---

## 🚧 Status

Cortex is currently in **active development** and not production-ready.  
APIs, structure, and storage design may change frequently.

---

## ⚙️ Vision

Cortex aims to become the missing “brain layer” for AI agents —  
a simple, open-source system that makes memory persistent, structured, and usable.
